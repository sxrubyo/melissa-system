## ⚡ **Start Selling in 3 Steps**

### **Step 1: Install Melissa (60 seconds)**

```bash
npm install -g melissa-ai
melissa --version
```

### **Step 2: Create Your First Agent (5 minutes)**

```bash
melissa persona create restaurant-receptionist
```

Edit `personas/restaurant-receptionist.txt`:

```
You are Sofia, the AI receptionist for "La Cucina Bella" Italian restaurant.

Your job:
- Answer questions about menu, hours, location
- Take reservations (collect: name, phone, date, time, party size)
- Send the menu PDF when asked
- Be warm, friendly, professional

Hours: Mon-Sun 11am-10pm
Location: 123 Main St, Miami, FL
Specials: Tuesday = 2x1 Pasta Night

When someone wants to book, say:
"Perfect! I'll need: your name, phone number, preferred date & time, and how many people."

Never:
- Give discounts without manager approval
- Confirm reservations (say "I'll pass this to our manager")
- Share financial information
```

### **Step 3: Deploy to Client (30 seconds)**

```bash
# Clone core to client folder
melissa sync --add /opt/melissa-restaurant-bella

# Client configures their own WhatsApp number in .env
cd /opt/melissa-restaurant-bella
cp .env.example .env
nano .env  # add WHATSAPP_PHONE, OPENAI_KEY

# Start their instance
python3 melissa.py
```

**Done.** Their WhatsApp number now has an AI receptionist. They update the persona file. You never touch credentials.

```
                     ┌──────────────────────────┐

                     │    ~/melissa-dev/         │
                     └──────────┬───────────────┘
                                │
                     ┌──────────▼───────────────┐

                     │   (Core Replication)      │
                     └──────────┬───────────────┘
                                │
        ┌───────────────────────┼───────────────────────┐
        │                       │                       │
┌───────▼────────┐    ┌────────▼────────┐    ┌────────▼────────┐
│  Client A      │    │  Client B       │    │  Client C       │
│  Restaurant    │    │  Salon          │    │  Real Estate    │
├────────────────┤    ├─────────────────┤    ├─────────────────┤
│ Their .env     │    │ Their .env      │    │ Their .env      │
│ Their persona  │    │ Their persona   │    │ Their persona   │
│ Their database │    │ Their database  │    │ Their database  │
│ Their WhatsApp │    │ Their Telegram  │    │ Their WhatsApp  │
└────────────────┘    └─────────────────┘    └─────────────────┘
```

### 🧠 **What Each Client Can Control (Without Touching Code)**

| File | What Client Edits | Example |
|:---|:---|:---|
| `.env` | API keys, phone numbers, tokens | `WHATSAPP_PHONE=+1234567890` |
| `personas/their-bot.txt` | Personality, rules, FAQ | "You are John, a real estate assistant..." |
| `melissa_agents/custom.py` | Custom skills (optional) | Integration with their CRM |

### 🔒 **What You Control (Core Updates)**

| File | What You Update | Why |
|:---|:---|:---|
| `melissa.py` | Engine improvements | Better performance, bug fixes |
| `melissa_brain_v10.py` | Memory upgrades | Longer context, better retention |
| `melissa_core/` | Conversation logic | Handle edge cases |
| `requirements.txt` | Dependencies | Security patches |

When you run `melissa sync`, **only these core files update**. Client data stays untouched.

## 🔥 **Installation (Production)**

### **Option 1: VPS (DigitalOcean, Linode, Vultr)**

```bash
# SSH into your server
ssh root@your-server-ip

# Install Melissa
npm install -g melissa-ai

# Create first client
melissa sync --add /opt/client-001

# Configure
cd /opt/client-001
cp .env.example .env
nano .env  # add WHATSAPP_PHONE, OPENAI_KEY

# Start with systemd (auto-restart on crash)
sudo nano /etc/systemd/system/melissa-client-001.service
```

**Service file:**
```ini
[Unit]
Description=Melissa AI - Client 001
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/client-001
ExecStart=/usr/bin/python3 /opt/client-001/melissa.py
Restart=always

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable melissa-client-001
sudo systemctl start melissa-client-001
```

### **Option 2: Docker (Easier)**

```bash
docker run -d \
  --name melissa-client-001 \
  -p 8000:8000 \
  -v /opt/client-001:/app \
  -e WHATSAPP_PHONE=+1234567890 \
  -e OPENAI_KEY=sk-... \
  melissa-ai:latest
```
